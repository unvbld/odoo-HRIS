HR Attendance Face Recognition

"This module enables employees to log their attendance by checking in and out using Face Recognition. It seamlessly integrates with the system to capture and record employee photographs during each check-in and check-out."

Installation

Before using this module, make sure you have installed the required dependencies:

                pip install face-recognition
                pip install dlib

    Screenshots

'Face Recognition Attendance' Button visibility.

First, you need to activate the 'Face Recognition Attendance' button, which will be visible in the Odoo navigation bar.

Employee Image Registration for Face Recognition

Click on the camera icon to capture the employee's image. This image will be used for future check-ins and check-outs to verify if the employee matches the recorded photo.

Attendance -> Checkin/Checkout

Log employee attendance by capturing check-ins and check-outs using face recognition.
This ensures accurate and secure attendance tracking, integrating seamlessly with employee records.

Face Recognition-Based Attendance Check-In and Check-Out

Clicking "Check In" captures the employee's photo and records their attendance as checked in.
Similarly, clicking "Check Out" captures the photo again and marks the attendance as checked out.

Storing Captured Images in Attendance Records

The images captured during check-in and check-out will be stored in the employee's attendance record under the 'Attendance' tab.
